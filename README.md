# In Loving Memory of Dada | श्रद्धांजलि

A memorial website in loving memory of Kailash Nath Bhargava (February 13, 1941 – March 4, 2026). Built by his family to preserve his legacy, stories, wisdom, and the values he lived by.

**Live site:** [mananb77.github.io/dada](https://mananb77.github.io/dada/)

## Pages

- **Home** — His life summary, family tree, journey timeline, quotes, values, photo gallery, and family voices
- **Life Story** (`/lifestory`) — Full biographical narrative in 13 chapters, from his birth in Beawar through his banking career and beyond
- **Family Stories** (`/stories`) — Anecdotes and longer stories shared by family during the remembrance call
- **From His Grandkids** (`/grandkids`) — Memories and reflections recorded by his grandchildren
- **Memories** (`/memories`) — Community memory board with Google Forms integration for submissions

## Structure

- `memorial-site/` — React/Vite memorial website
- `photos/` — Photos of Dada
- `Dada Lifestory Document.md` — Lifestory source content
- `Remembering Dada Grandson.md` — Grandkids conversation transcript

## Tech Stack

- React 19 + Vite
- React Router (client-side routing)
- Google Fonts (Cormorant Garamond, Inter, Noto Sans Devanagari)
- Bilingual English/Hindi throughout
- Deployed on GitHub Pages at `/dada/`

## Development

```bash
cd memorial-site
npm install
npm run dev      # local dev server
npm run build    # production build
```
